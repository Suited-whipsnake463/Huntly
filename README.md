## 🚀 Overview

Huntly is an end-user app for lead generation on Windows. It helps you find business leads from Google Maps, read reviews for pain signals, and run email follow-ups that match each lead.

Use it to build lead lists, spot buying intent, and send outreach that feels relevant. The app keeps the process in one place, so you do not need to move between many tools.

## 💻 What Huntly Does

Huntly helps you:

- Search for local businesses on Google Maps
- Collect lead data into one list
- Read reviews for signs of problems or needs
- Use AI to pull out pain points from review text
- Send drip email campaigns based on lead details
- Track outreach work in a simple dashboard

## 📋 How to Use Huntly

### 1. Search for leads

Enter a city, service, or business type. Huntly checks Google Maps style business data and builds a lead list for you.

### 2. Review business signals

Open a lead and read its reviews. Huntly looks for words that point to problems, delays, weak service, or missed chances. These pain signals help you choose better leads.

### 3. Create a message

Use the lead details and review insights to write a message that fits the business. Keep it short and direct.

### 4. Start a drip campaign

Pick how many emails to send and how far apart to send them. Huntly can run a set sequence so each lead gets follow-up messages over time.

### 5. Track replies and progress

Use the dashboard to see which leads were contacted, which ones replied, and which ones still need follow-up.

## 🧰 Troubleshooting

If Huntly does not start, try these steps:

- Make sure the download finished
- Extract the zip file again
- Right-click the app and choose Run as administrator
- Restart your PC
- Check that your Windows version is up to date
- Close other apps that use a lot of memory
- Try opening the app from the main folder, not a shortcut

If the browser does not open, check your default browser and confirm that it is installed and working.

If email sending does not work, check:

- Your email address and password
- Your connection to the internet
- Any inbox security settings
- Whether your provider blocks app sign-in
